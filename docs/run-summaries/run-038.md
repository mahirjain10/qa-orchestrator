## What to watch for
- When `BROWSER_MODE=real`, the browser registry is used for both LLM tools and executor — it has `evaluate`, `get_html`, etc.
- When `BROWSER_MODE` is not set (default), `MockToolRegistry` is used — LLM only sees tools in `getDefaultLLMTools()`.
- The two tool lists must always stay in sync.

## Key decisions
- Removed `get_html` and `evaluate` from default LLM tools rather than adding them to MockToolRegistry — these are browser-specific tools that don't make sense in simulation mode.
