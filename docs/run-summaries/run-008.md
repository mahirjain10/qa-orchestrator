## What went wrong
- Data loss: ArtifactStore index wasn't being persisted properly because the `persistIndex` function was an empty stub.
- Ghost files: Empty `index.json` wasn't being deleted.

## What to watch for
- Always test storage systems with empty data scenarios, as edge cases expose the worst bugs.
- Log errors to standard error (stderr) at minimum; never silently swallow them in a `continue` loop.
- Test file deletion scenarios (both when files exist and when the index is empty).

## Key decisions
- Always call `persistIndex` after any index modification.
- Explicitly delete `index.json` using `os.Remove` when the artifact index becomes empty.
- Log unmarshal errors with `fmt.Fprintf(os.Stderr, ...)` for debugging.
