## What to watch for
- **View type vs ComponentID**: The old `ComponentID` enum is gone. All view references now use `View` type (`ViewDashboard`, `ViewFlows`, `ViewTraces`, `ViewReport`).
- **Sidebar focus vs slot focus**: Navigation now uses `sidebarFocus` boolean instead of `activeSlot`. When sidebar is focused, ↑↓ cycle views. When content is focused, ↑↓ navigate the active component's list.
- **No more maximized state**: The `maximized`/`maximizedSlot` fields are gone. Full-screen views are handled by the view system itself.
- **Status enums**: `RunState` has no `Passed` or `Retrying` — use `Completed` for run completion. `FlowState` has `SkippedUpstream` and `SkippedUser` but no plain `Skipped`.
